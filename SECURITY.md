# Security Policy

## Supported Versions

This repository contains research tooling. It is provided as-is.

## Reporting a Vulnerability

If you believe you have found a security issue in these tools (not in a third-party target),
please contact the maintainers privately.

- Email: security@itresit.es

Please include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Suggested fix (if available)

Do not open a public issue with exploit details.
